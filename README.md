# NexusHR Day 1 Setup

This workspace contains the Day 1 backend foundation for the NexusHR platform.

Modules created:

- `api-gateway`
- `auth-service`
- `employee-service`
- `payroll-service`
- `common`

The setup uses:

- Java 21
- Spring Boot 3.3
- Maven multi-module structure
- `application.properties` in each service

Git was intentionally not used as requested.
